# BLACKBOX SQUADRON Project Status

## Current build

**v0.1.4**

## Current state

The game is a playable single-file HTML arcade shooter prototype with:

- canvas rendering
- keyboard controls
- touch controls
- mobile viewport scaling
- player movement
- player fire
- bomb mechanic
- enemies
- enemy bullets
- spread shot pickup
- obstacles / debris
- score
- high score via localStorage
- lives
- level progression
- start / pause / game over screens

## Known design gap

The current reward loop is too flat. The game only drops Spread Shot. There are no repair pickups, bomb pickups, shield pickups, rapid fire, rail, or drone pickups yet.

## Recommended next patch

**v0.2 — Expanded Power-Ups + Level Director**

Do not add bosses yet. Fix reward variety and level personality first.
