# GitHub Upload Checklist

## Required files

- [ ] `index.html` exists in the repo root
- [ ] `README.md` exists in the repo root
- [ ] `.nojekyll` exists in the repo root
- [ ] `docs/` folder is uploaded

## GitHub Pages setup

- [ ] Repository created
- [ ] Files uploaded to root, not nested inside another folder
- [ ] Settings → Pages opened
- [ ] Source set to **Deploy from a branch**
- [ ] Branch set to **main**
- [ ] Folder set to **/root**
- [ ] Pages URL tested on desktop
- [ ] Pages URL tested on phone

## Smoke test

- [ ] Start screen appears
- [ ] Keyboard start works
- [ ] Touch start works on phone
- [ ] Player moves
- [ ] Player fires
- [ ] Bomb works
- [ ] Enemies spawn
- [ ] Game over works
- [ ] Restart works
