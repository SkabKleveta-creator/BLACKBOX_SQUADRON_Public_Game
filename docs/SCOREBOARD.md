## About This Scoreboard

This file is reserved for manually verified public scores.

The game may store local high scores in each player's browser, but this public scoreboard is intended for community tracking through manual submission and review.

The game should **not** write directly to this file from browser JavaScript.

No GitHub tokens, API keys, or write credentials should ever be embedded in the game source.

## Submission Format

To submit a score, provide:

```text
BLACKBOX SQUADRON SCORE SUBMISSION

Pilot Name / Initials:
Score:
Level Reached:
Game Version:
Screenshot Attached: Yes / No

```

## Submission Rules

- Scores should include a screenshot when possible.
- Scores should include the game version.
- Scores should include level reached.
- Pilot names should stay short and arcade-style.
- Joke names are fine, but keep it clean.
- Public scores are manually reviewed before being added.

## Notes for Maintainer

Recommended workflow:

1. Player submits score through GitHub Issues, Facebook comment, Discord, text, or direct message.
2. Maintainer verifies screenshot or score claim.
3. Maintainer updates this file manually.
4. Highest scores stay sorted from highest to lowest.

Future option:

- Add GitHub Issue template for score submissions.
- Add a screenshot requirement.
- Add a version field.
- Add a simple “verified by” field.
